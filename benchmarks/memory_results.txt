=== Сравнение потребления памяти: Go Gin vs FastAPI ===

Методика:
- Замер памяти (RSS) до и после нагрузки в 200 запросов (ab -n 200 -c 10)
- Использован модуль psutil для Python

Результаты:

Go Gin:
  Память до нагрузки: 32.16 MB
  Память после нагрузки: 32.16 MB
  Разница: 0 MB
  Утечек памяти: не обнаружено

FastAPI:
  Память до нагрузки: 49.27 MB
  Память после нагрузки: 49.27 MB
  Разница: 0 MB
  Утечек памяти: не обнаружено

Выводы:
- FastAPI потребляет больше памяти (~49 MB против ~32 MB у Go Gin)
- Оба сервиса стабильны под нагрузкой, утечек памяти нет
- Go Gin более эффективен по памяти
